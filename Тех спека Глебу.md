
Примерный флоу по обработке запроса

```mermaid
sequenceDiagram
	participant Rabbit as RabbitMQ
	participant App as RunApp
	participant Docker as Docker API
	participant Db as Master Data DB
	
	Rabbit->>App: получем сообщение(CodeExecutionRequest)
	App->>Db: идем за тестами в Мастер базу по id задачи
	Db-->>App: получаем данные тестов
	App->>Docker: ищем свободный контейнер с нужным языком программирования
	Docker-->>App: возвращает ответ
	alt контейнер не доступен
		App->>App: ждет освобождения контейнера
	end
	App->>Docker: записывает код в контейнер
	Docker-->>App: успешная загрузка
	App->>Docker: запуск несколько тестов через многопоток
	Docker-->>App: возвращает результат тестов
	alt есть упавшие тесты
		Note over App: Статус failed, id упавших тестов
	else все тесты прошли
		Note over App: Статус accepted
	end
	App->>Rabbit: отправляем сообщение о статусе(CodeExecutionResult)
```

```csharp
public class CodeExecutionRequest
{
    public string Code { get; set; }
    public string Language { get; set; }
    public string Tests { get; set; }
    public string CorrelationId { get; set; }
    public Guid PackageId { get; set; }
}
```

```csharp
public class CodeExecutionResult
{
    public int PassedTests { get; set; }
    public int FailedTests { get; set; }
    public List<FailedTest> FailedTests { get; set; }
    public string CorrelationId { get; set; }
    public Guid PackageId { get; set; }
}

public struct FailedTest
{
	public Guid TestId { get; set; }
	public string Reason { get; set; }
}
```
